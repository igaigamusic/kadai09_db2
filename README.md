## ⑤工夫した点・こだわった点

- 今まで、患者さんが病院を受診した際の受付画面を作っていましたが、データの更新・削除の必要がありませんでした
- このため、データの更新・削除を行う可能性があるユーザー登録画面を作成しました
- 誕生日の入力はinput typeをdateにして入力しやすくしています

## ⑥難しかった点・次回トライしたいこと(又は機能)

- 最初ローカルで完成させてしまい、さくらのサーバーにデータベースを移行するのに時間がかかりました
- 一度FileZillaでフォルダをアップロードした後に、本番環境でページの遷移が正常に動かず、再度フォルダをアップロードしたらフォルダの中にフォルダができてしまい苦労しました
